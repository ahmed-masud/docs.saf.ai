sāf.ai Cybernetic Engram File System Documentation¶

Quick Start¶

Select your target system:

Cut and paste the following into a terminal to begin download and installation:

curl -s https://saf.ai/downloads/stable/cefs-init.sh | sudo bash

This should look something like the following:

Note

Native windows support is coming soon, meanwhile you can use Samba to export CEFS protected partition to your Windows Network. Since all the protection is built into the file system you don’t need any agents on your Microsoft Windows® end-points, and still get protection of any data stored on the CEFS partition.

Continue on to Installation for in-depth instructions or Test Instructions to test the new CEFS installation.
